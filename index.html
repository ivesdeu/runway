Connect once: your numeric Property ID and a service account JSON key. They stay on the API server until you disconnect — no Google sign-in each visit.

Find Property ID Create JSON key
Quick setup
  1. Open Google Cloud → Service accounts (pick the same project as GA4 if linked).
  2. Create a service account → Keys → Add key → JSON and download the file. In the same Google Cloud project, open APIs & Services → Library and enable Google Analytics Data API (required for reports).
  3. In GA4 → Admin → Property access management → + → add the exact client_email from your JSON with Viewer or Analyst on this property. (Grant access here in Analytics, not only under Google Cloud → IAM.)
  4. In GA4 → Admin, use the property dropdown in the left sidebar, open Property details, then find the numeric Property ID in your account profile (top right; digits only).
  5. Paste the Property ID below and upload or paste the JSON, then click Connect GA4.
